# test-repo

Test repository for meta-repo system.

## Dependencies

- [test-lib](https://github.com/Aeropaq/test-lib) - Testing utilities library

## Installation

```bash
npm install
```

## Build

```bash
npm run build
```

## Development

```bash
npm run dev
```

## Meta-Repo System

This repository is part of a meta-repo system and includes a `.meta-manifest.json` file for dependency management.
